Pseudo server application

Use a REST client ( e.g. Advanced REST client ) to make REST API calls.

/products

method URL parameters (JSON) result (JSON)
GET 
/products
 returns array of all products
GET 
/products/:id
 returns matched product entry
POST 
/products/create
 
{ product: string, ?quantity: number }
 creates product entry and returns resulting entry
PUT 
/products/:product_id/update
 
{ ?product: string, quantity: number }
 updates product entry and returns resulting entry
DELETE 
/products/:id/delete
 deletes matching product entry
GET 
/products/search/:query
 returns an array of matching product entries

/customers

method URL parameters (JSON) result (JSON)
GET 
/customers
 returns array of all customers
GET 
/customers/:id
 returns matched customer entry
POST 
/customers/create
 
{ name: string, ?email: string }
 creates customer entry and returns resulting entry
PUT 
/customers/:id/update
 
{ ?name: string, ?email: string }
 updates customer entry and returns resulting entry
DELETE 
/customers/:id/delete
 deletes matching customer entry
GET 
/customers/search/:query
 returns an array of matched customer entries
GET 
/customers/:id/get-products
 returns all products owned by matched customer
GET 
/customers/:id/get-products/:query
 returns an array of matching products owned by matched customer
POST 
/customers/:id/add-product
 
{ product: string, quantity: number }
 transfers the defined product to the matched customer
